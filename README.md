# Educational-Security-Animations
Python based educational tool demonstrating cryptographic concepts through animations and quizzes, focusing mainly on GCM and CTR modes of operation.

# Features
- Custom made animations
- Quizzes
- Glossary
- Accessibility features - custom text sizing, light/dark modes
- Local progress tracking

# Concepts covered
- GCM encryption & decryption
- CTR encryption & decryption
- XOR
- The security problem & the need for modes of operation

# Technologies used
- Python
- Tkinter

# Skills used
- Software development
- UI design
- Educational design
- Cryptography
- Report writing

# Requirements
Windows 11 system with Python installed.
Tkinter library installed

To run application, execute Main.py.
